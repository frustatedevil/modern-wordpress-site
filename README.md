# Modern WordPress Website

A production-ready WordPress installation with modern standards and best practices.

## Features

- **Latest WordPress** - Always up-to-date with the latest version
- **Optimized Performance** - Enhanced PHP settings and caching
- **Security Hardened** - SSL enforcement and secure configurations
- **WP-CLI Included** - Command-line management tools
- **Docker-based** - Easy deployment and scaling

## Tech Stack

- WordPress (Latest)
- MySQL Database
- Docker & Docker Compose
- Railway Platform

## Configuration

Environment variables required:
- `MYSQL_HOST` - Database host
- `MYSQL_USER` - Database user
- `MYSQL_PASSWORD` - Database password
- `MYSQL_DATABASE` - Database name
- `RAILWAY_PUBLIC_DOMAIN` - Your site URL

## Deployment

Deployed on Railway with automatic SSL and custom domain support.

## Maintenance

This site follows market standards:
- Regular WordPress core updates
- Plugin security updates
- Performance optimization
- SEO best practices
- Mobile responsiveness
- HTTPS enforcement

## Access

After deployment, access your WordPress admin at: `https://your-domain.com/wp-admin`

Default setup will prompt you to create an admin account on first visit.
